## ðŸ“¦ Installation
VueSock is GSAP-version agnostic and should work with any version you throw into your project. 

```sh
// with npm
$ npm install gsap vue-sock

// or with yarn
$ yarn add gsap vue-sock

// or with pnpm
$ pnpm add gsap vue-sock
```
